Git e GitHub

A função do Git e GitHub é proporcionar a facilitação do "versionamento de códigos", mas ele possui inumeras funções tanto particulares e pessoais, quanto corporativas.

Principais Funções:

  • Hospedagem de Repositórios
  • Colaboração em Equipes
  • Rastreamento de Problemas e Projetos
  • Criação de Forks e Branches
  • Versionamento de Códigos

Configurando o Git

Abra o Git Bash na sua pasta de destino das informações e siga as seguintes etapas:

  • $git config --global user.name "nome_do_usuario"
  • $git config --global user.email "email@dousuario"

Esses comandos servem para configurar as informações globais do usuário. Caso erre em algo, basta dar o comando novamente.
